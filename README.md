# airphoto_cityhall_echizen

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

This project provides a tool for capturing aerial photographs of Echizen City Hall using a drone.

## Features
- Automated drone flight path planning
- Supports multiple camera angles and altitudes
- Outputs high-resolution aerial photographs

## Requirements
- DJI Phantom 4 Pro drone
- Python 3.x
- OpenCV library

## Usage
1. Install the required Python dependencies:
   ```
   pip install -r requirements.txt
   ```
2. Configure the flight plan parameters in the `config.py` file.
3. Run the `main.py` script to execute the automated photography mission.
